## Cómo ejecutar los notebooks

### En VS Code

1. Instala la extensión de Python.
2. Abre la carpeta del proyecto en VS Code.
3. Abre un archivo `.ipynb` desde `notebooks/`.
4. Selecciona el kernel de Python.

### En Jupyter Notebook

1. Asegúrate de tener Jupyter instalado.
2. Ejecuta `jupyter notebook` en la raíz del proyecto.
3. Abre el notebook deseado desde `notebooks/`.

### En Google Colab

Abre cualquier notebook con los botones “Open in Colab”.
